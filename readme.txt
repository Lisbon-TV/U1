#U1 Bankas REACT

REACT banko aplikacijos versija


Banką sudaro puslapis, kuriame yra:

Sąskaitų sąrašas su Vardu, Pavarde, Sąskaitos suma, Mygtuku “ištrinti”,  
laukeliu vertei įrašyti ir dviem mygtukais tam laukeliui: 
“pridėti lėšų” ir “nuskaičiuoti lėšas”;

Naujos sąskaitos sukūrimas (įvedami duomenys: vardas ir pavardė).

Paaiškinimai ir reikalav`imai:

Nauja sąskaita sukuriama su pradine 0 suma, o lėšos pridedamos/nuimamos sąraše įvedant sumą
ir spaudžiant atitinkamą mygtuką.
Sąskaitos, kurioje yra lėšų ištrinti neturi būti galima.
Sąskaitoje likusi suma negali būti minusinė. 
Sąskaitas saraše rūšiuoti pagal savininko pavardę.
Duomenų bazė - LocalStorage.
